World Wide Web
  • Collection of HTML documents
    • which stands for HyperText Markup Language
  • Marjo pieces include:
    • Browser, Internet, and Server
      • Browser makes requests to servers via the internet
HTML
  • Heart of the web and is made up of:
    • Text Content: which is what you see
    • Markup: which is what it looks like
    • References: which are images and ideas
    • Links: to other pages
Mark Up
  • Contain the opening tag, the contents, and then the closing tag
    • In its entirety, this is called an element
HTML Attributes
  • Open tag, attribute, value, contents, and then close tag
    • Attribute is always equal to a value
Tags
  • Close Tags are used when content is usesd, while void tags are used when there is no content
  • Bold Tags
    • Called "B-Tags"
  • Itallic Tags or Emphasis Tags
    • Called "Em-Tags"
  • Image Tags
    • Image tags are void tags because they do not contain content
    • Image tags contain attributes and the url for the image
  • White Space
    • Used to force HTML to have another line
    • A Break Tag is a void tag and is inline
    • A Paragraph Tag has content and is block
Inline v. Block
  • Inline
    • Ends a line
    • B-Tags, Em-Tags, Image Tags
  • Block
    • Makes invisible block around HTML
    • Paragraph Tags
